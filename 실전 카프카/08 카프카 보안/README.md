# 카프카 보안 
 
카프카는 대다수 기업의 중앙 데이터 파이프라인 허브로서 역할을 수행하고 있다.        
카프카는 기업 내부의 시스템으로 주로 활용하기 때문에 대부분 사내 보안에 대해서는 큰 신경을 쓰지는 않는다.     
간혹 RabbitMQ와 같은 오픈소스에서는    
자체적으로 기본적인 보안 기능이 포함된 경우도 있어서 관리자가 원한다면 초기 구성부터 즉시 보안을 적용할 수 있지만,     
카프카가 제공하는 보안 기능은 카프카 설치시에 자동으로 설정되지 않기 때문에 보안이 필요한 상황이라면 관리자가 직접 적용해야한다.
